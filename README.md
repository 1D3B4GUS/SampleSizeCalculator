# Kalkulator Ukuran Sampel · Sample Size Calculator

**Bahasa Indonesia** | [English below](#english)

---

## 🇮🇩 Indonesia

**Kalkulator Ukuran Sampel** adalah aplikasi web interaktif untuk menghitung dan membandingkan besar sampel menggunakan lima rumus yang umum digunakan dalam penelitian psikologi dan pendidikan di Indonesia.

### 🔗 Akses Aplikasi

> **https://1d3b4gus.github.io/SampleSizeCalculator/**

Berjalan langsung di peramban — tidak perlu instalasi.

### ✨ Fitur Utama

- **5 rumus sekaligus** — Cochran, Krejcie–Morgan, Isaac & Michael, Nomogram Harry King, dan Slovin
- **Gerbang asumsi wajib** — 4 syarat diperiksa sebelum hasil ditampilkan, mencegah penyalahgunaan rumus
- **Pohon keputusan** — panduan untuk peneliti yang kondisi penelitiannya tidak memenuhi syarat rumus (SEM, CFA, nonprobability, kualitatif)
- **Sampling Decision Report** — dokumen transparansi metodologis lengkap dengan flowchart JPG siap lampir ke naskah
- **Kalimat siap-tempel** — teks bagian Metode yang sudah memuat semua parameter yang dibutuhkan
- **Bilingual** — Indonesia dan English dalam satu file
- **Gratis & open** — satu file HTML, tidak ada server, tidak ada data yang dikumpulkan

### 📋 Kondisi Penggunaan

Rumus-rumus dalam aplikasi ini hanya valid pada kondisi:
1. Sampel diambil secara **SRSWOR** (Simple Random Sampling Without Replacement)
2. Jumlah populasi **diketahui dan terbatas**
3. Analisis berupa **proporsi atau rata-rata satu kelompok**
4. Tujuan penelitian adalah **presisi estimasi**, bukan uji hipotesis

Untuk kondisi lain, gunakan Tab 2 · Pohon Keputusan di dalam aplikasi.

## <a name="english"></a>🇬🇧 English

**Sample Size Calculator** is an interactive web application for calculating and comparing sample sizes using five formulas widely cited in psychology and education research.

### 🔗 Access the App

> **https://1d3b4gus.github.io/SampleSizeCalculator/**

Runs directly in any browser — no installation required.

### ✨ Key Features

- **5 formulas at once** — Cochran, Krejcie–Morgan, Isaac & Michael, Harry King Nomogram, and Slovin
- **Mandatory assumption gateway** — 4 conditions checked before results are shown, preventing formula misuse
- **Decision tree** — guidance for researchers whose conditions fall outside the scope of these formulas (SEM, CFA, nonprobability, qualitative)
- **Sampling Decision Report** — full methodological transparency document with downloadable JPG flowchart for manuscript attachment
- **Ready-to-paste Methods statement** — pre-written text with all required parameters
- **Bilingual** — Indonesian and English in a single file
- **Free & open** — single HTML file, no server, no data collected

### 📋 Conditions of Use

The formulas in this application are only valid when:
1. Sample is drawn by **SRSWOR** (Simple Random Sampling Without Replacement)
2. Population size is **known and finite**
3. Analysis concerns a **proportion or mean of one group**
4. Research goal is **precision of estimation**, not hypothesis testing

For other conditions, use Tab 2 · Decision Tree within the application.
